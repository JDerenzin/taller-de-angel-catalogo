# Catálogo Express - MVP

Este es un catálogo minimalista diseñado para reducir la fricción en ventas por Instagram. 
Permite elegir opciones, genera un resumen automático, lo copia al portapapeles y redirige al cliente directamente al DM del vendedor.

## 🚀 Flujo del Usuario
1. **Selección:** El usuario elige productos del catálogo.
2. **Copiado:** Al finalizar, el sistema copia el resumen formateado (UTF-8 + Emojis).
3. **Redirección:** Se abre el enlace `ig.me` para que el usuario solo tenga que "Pegar" y enviar.

## 🛠️ Tecnologías
- HTML5 / CSS3 (Custom Properties & Grid)
- Vanilla JavaScript (ES Modules)
- Desplegado en: [Vercel](https://vercel.com)

## 📦 Instalación y Deploy
No requiere dependencias. 
Para probarlo localmente, puedes usar cualquier servidor estático (como 'Live Server' en VS Code).

Para desplegar en Vercel:
1. Sube este repo a GitHub.
2. Conecta el repo en el dashboard de Vercel.
3. ¡Listo!
